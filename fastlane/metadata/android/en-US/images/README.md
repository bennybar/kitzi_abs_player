# F-Droid Images

This directory contains the required images for F-Droid publication.

## Required Files

- `icon.png` - App icon (512x512px recommended)
- `phoneScreenshots/` - Phone screenshots directory

## Screenshots

Add phone screenshots to the `phoneScreenshots/` directory. Recommended:
- 3-5 screenshots showing key features
- 1080x1920px or similar phone aspect ratio
- PNG or JPEG format
- Show main screens: home, player, settings, etc.
